# HTML `<bdo>` 标签

##### 标签：`<bdo>`

**说明：**定义一段内容的文本方向，一般用于与默认方向相反的方向。

**属性：**

| 属性  | 值               | 说明                                                         |
| ----- | ---------------- | ------------------------------------------------------------ |
| `dir` | `ltr`<br />`rtl` | 规定标签内的文本方向。<br />`ltr`：从左向右<br />`rtl`：从右向左 |

**示例：**略。
