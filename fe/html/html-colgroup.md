# HTML `<colgroup>` 标签

##### 标签：`<colgroup>`

**说明：**对表格中的所有列进行格式化。`<colgroup>` 只能在 `<table>` 中使用，且在 `<caption>` 标签之后，在 `<thead>`、`<tbody>`、`<tfoot>`、`<tr>` 标签之前。如果需要对特定的列执行特殊的格式化，可以配合 `<col>` 标签使用。

**属性：**

| 属性   | 值   | 说明               |
| ------ | ---- | ------------------ |
| `span` |      | 规定列组横跨的列数 |

**示例：**略。
