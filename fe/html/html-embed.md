# HTML `<embed>` 标签

**标签：`<embed>`**

**说明：**定义一个容器，用来嵌入外部应用或程序。

**属性：**

| 属性     | 值   | 说明                     |
| -------- | ---- | ------------------------ |
| `height` |      | 定义嵌入内容的高度       |
| `src`    |      | 定义被嵌入内容的 URL     |
| `type`   |      | 规定嵌入内容的 MIME 类型 |
| `width`  |      | 规定嵌入内容的宽度       |

**示例：**略。
