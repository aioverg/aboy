#### 数值类型

##### 字面量

1. 整数：`1234`（无大小限制）
2. 浮点数：`1234.0`
3. 复数：`2 +2j `

##### 运算符

| 运算符                                     | 描述                                         |
| ------------------------------------------ | -------------------------------------------- |
| `yield x`                                  | 生成器函数 `send` 协议                       |
| `lambbda args: expression`                 | 创建匿名函数                                 |
| `x if y else z`                            | 三元运算符，`x` 为真时运算 `y`，否则运算 `z` |
| `x or y`                                   | 逻辑或，`x` 为假时运算 `y`                   |
| `x and y`                                  | 逻辑与，`x` 为真时运算 `y`                   |
| `not x`                                    | 逻辑非，`x` 的反面                           |
| `x in y`、 `x not in y`                    | 成员关系（用于可迭代对象、集合）             |
| `x is y` 、 `x is not y`                   | 同一性检查                                   |
| `x < y` 、 `x <= y` 、 `x > y` 、 `x >= y` | 大小比较                                     |
| `x == y` 、 `x != y`                       | 等价性检查                                   |
| `x | y`                                    | 按位或、集合并集                             |
| `x ^ y`                                    | 按位异或、集合差集                           |
| `x & y`                                    | 按位与、集合交集                             |
| `x << y` 、 `x >>y`                        | 将 `x` 左移或右移 `y` 位                     |
| `x + y`                                    | 加法、拼接                                   |
| `x - y`                                    | 减法、集合差集                               |
| `x * y`                                    | 乘法、重复                                   |
| `x % y`                                    | 求余数、格式化                               |
| `x / y` 、 `x // y`                        | 真除法、向下取整除法                         |
| `-x` 、`+x`                                | 取负、取正                                   |
| `~x`                                       | 按位非（取反码）                             |
| `x ** y`                                   | 幂运算                                       |
| `x[i]`                                     | 索引                                         |
| `x[i:j:k]`                                 | 分片                                         |
| `x(···)`                                   | 调用（函数、方法、类等）                     |
| `x.attr`                                   | 属性引用                                     |
| `(···)`                                    | 元组、表达式、生成器表达式                   |
| `[···]`                                    | 列表、列表推导                               |
| `{···}`                                    | 字典、集合、集合与字典推导                   |
