#### 元组、文件、其核心他类型

##### 元组常用操作

| 操作                         | 说明                       |
| :--------------------------- | -------------------------- |
| `()`                         | 元组                       |
| `T[index]`                   | 索引                       |
| `T[i:j]`                     | 分片                       |
| `len(T)`                     | 长度                       |
| `+`                          | 拼接                       |
| `*`                          | 重复                       |
| `for item in T: print(item)` | 迭代                       |
| `item in T`                  | 成员检测，返回布尔值       |
| `T.index(item)`              | 成员检测，返回索引或者报错 |
| `T.count(item)`              | 返回成员在元组中出现的次数 |
| `list(T)`                    | 将元组转换成列表           |
| `tulpe(L)`                   | 将列表转换成元组           |

元组的特点：

- 任意对象的有序集合
- 通过偏移量取值
- 是不可变序列

注意：元组由 `()` 包裹，各项使用 `,` 分隔，但如果元组只有一项时很容易和表达式混淆，所以在元组只有一项时也应该在项后面加上`,`以作区分。
